# EyeTrackerOSC

VRChatのOSCで目を動かせるようになる。

下記対応したデバイスが必要

XR_FB_eye_tracking_social

XR_FB_face_tracking

例：現状 Quest Proのみ

※さっさとXR_MND_headlessとPCモードでXR_EXT_eye_gaze_interaction対応して！

## Build from source

openxr_loader.lib to lib

Open with VisualStudio

Build

## Option

ip port sleepTime
